## Navigation Methods

### 1. Top Menu Bar

The persistent menu bar at the top provides global navigation:

**Always Visible**:
- Traxxia logo (return to dashboard)
- User menu (profile, settings, logout)
- Language selector
- Traxxia Academy link

![Top menu bar navigation](ALT_TEXT_PLACEHOLDER)

### 2. User Dropdown Menu

Click your profile icon (top-right) to access:

| Link | Takes You To | Available To |
|------|--------------|--------------|
| **Dashboard** | Main business listing | All (except Super Admin) |
| **Traxxia Academy** | This documentation site | Everyone |
| **Super Admin Panel** | System administration | Super Admin only |
| **Admin** | Company/user management | Company Admin, Super Admin |
| **Logout** | Login page (ends session) | Everyone |

### 3. In-App Navigation

Within a business, use:
- **Tabs** to switch between sections
- **Breadcrumbs** to track your location
- **Back buttons** to return to previous views
- **Sidebar menus** for subsections

## Breadcrumb Navigation

Breadcrumbs show your current location and provide quick navigation back:

### Example:
```
Dashboard > My Business > AI Questionnaire > Vision Module
```

**How to Use**:
- Click any breadcrumb link to jump to that level
- Last item (current page) is not clickable
- Breadcrumbs update automatically

![Breadcrumb navigation example](ALT_TEXT_PLACEHOLDER)

> [!TIP]
> Breadcrumbs are especially useful when you're deep in a workflow and need to quickly return to a higher level.

## Navigation Tips & Tricks

### 1. Use the Logo for Quick Reset
Clicked too deep into menus? Click the Traxxia logo to instantly return to your dashboard.

### 2. Breadcrumbs Are Your Friend
Don't rely on the browser's back button. Use breadcrumbs for cleaner, more predictable navigation.

### 3. Right-Click to Open in New Tab
Want to compare two sections? Right-click business cards or links and "Open in new tab."

### 4. Bookmark Frequently Used Pages
If you work primarily in one business, bookmark that URL for direct access.

### 5. Use Browser History Wisely
Traxxia maintains clean URLs that work with browser history. The back button generally works as expected.

## Navigation Flow Examples

### Example 1: Creating and Working on a Business


## Common Navigation Mistakes

### Mistake 1: Using Browser Back Button Excessively
**Problem**: Can lead to unexpected page states  
**Solution**: Use breadcrumbs and in-app navigation

### Mistake 2: Not Realizing You're in a Business
**Problem**: Trying to access dashboard features while inside a business  
**Solution**: Check breadcrumbs for your location; click logo to return

### Mistake  3: Losing Track of Changes
**Problem**: Making edits and navigating away  
**Solution**: Look for auto-save indicators; confirm saves before navigating

### Mistake 4: Not Using the User Menu
**Problem**: Can't find Traxxia Academy or admin features  
**Solution**: Everything you need is in the user menu (profile icon, top-right)

## Navigation Troubleshooting

### Issue: Can't Find a Feature

**Solution**:
1. Check your user role - you may not have permission
2. Review [Understanding User Roles](./03-understanding-user-roles.md)
3. Ask your administrator for access

### Issue: Lost in the Interface

**Solution**:
1. Check the breadcrumbs to see where you are
2. Click the Traxxia logo to return to dashboard
3. Start your workflow again from a known location

### Issue: Page Won't Load

**Solution**:
- Refresh the page (F5)
- Clear browser cache
- Try a different browser
- Check internet connection

### Issue: Navigation Feels Slow

**Possible Causes**:
- Large dataset loading
- Slow internet connection
- Too many browser tabs open

**Solution**:
- Close unused tabs
- Wait for page to fully load before navigating
- Check network speed

## Navigation Best Practices

### 1. Plan Your Workflow
Before diving in, know what you want to accomplish:
- Answering questionnaire in one business?
- Reviewing analyses across multiple businesses?
- Creating projects?

### 2. Work in Logical Sequence
Follow the natural flow:
```
Create Business ‚Üí Complete Questionnaire ‚Üí Review Analysis ‚Üí Create Projects
```

### 3. Minimize Context Switching  
Finish work in one section before moving to another. Constantly jumping between sections slows you down.

### 4. Use Multiple Browser Tabs
If you need to compare:
- Open businesses in separate tabs
- View analysis and questionnaire side-by-side
- Keep dashboard always open in one tab

### 5. Bookmark Your Workspace
If you work on the same business daily, bookmark it directly instead of always going through the dashboard.
