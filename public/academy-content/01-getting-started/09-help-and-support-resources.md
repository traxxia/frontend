## Troubleshooting Common Issues

### Login Problems

**Can't log in?**
1. Verify email address is correct
2. Try password reset
3. Check for verification email (check spam folder)
4. Clear browser cache and cookies

[Learn more about account issues](./02-creating-an-account.md#troubleshooting)

### Performance Issues

**Platform running slowly?**
1. Check internet connection
2. Close unnecessary browser tabs
3. Clear browser cache
4. Try a different browser
5. Disable browser extensions temporarily

### File Upload Issues

**Can't upload documents?**
1. Check file size (max 10MB for most uploads)
2. Verify file format is supported
3. Ensure stable internet connection
4. Try different browser

[View supported file formats](../05-financial-analysis/02-supported-file-formats.md)

### Analysis Not Generating

**Waiting too long for analysis?**
1. Check that questionnaire is complete
2. Verify all required phases are finished
3. Refresh the page
4. Check for error messages
5. Contact support if issue persists
