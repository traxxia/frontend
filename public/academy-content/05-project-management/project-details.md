# Project Details & Viewing

Clicking on any project card or using the edit menu opens the **Project Details** page. This is where you document the specific "Strategic Bet" you are making.

## 1. Strategic Core
This section defines the heart of the project:
- **Elevator Pitch**: A one-sentence summary of the project.
- **Why This Matters**: The business value and rationale.
- **Strategic Decision**: The core bet you are making.
- **Key Assumptions Tested**: A list of things that must be true for this project to succeed.
- **Success Criteria (Continue If)**: Metrics that signal the project is working.
- **Kill Criteria (Stop If)**: Warning signs that the project should be abandoned.

![Project Details Layout](/academy-screenshots/projects/project-details-view.png)

## 2. Strategic Context
These fields are used by the AI to help build your prioritization matrix:
- **Impact**: The potential upside (High, Medium, Low).
- **Effort**: The estimated resources required (Small, Medium, Large).
- **Risk**: The level of uncertainty.
- **Strategic Theme**: Category alignment (Growth, Efficiency, Innovation, etc.).

## 3. Additional Metadata
For advanced planning, you can document:
- **Constraints**: "Non-negotiables" for the project.
- **Dependencies**: Other projects or resources needed.
- **Out of Scope**: Clear boundaries to prevent scope creep.
- **Estimated Timeline & Budget**: Logical constraints for the initiative.

## 4. AI Assistant Context
The AI sidebar is your constant companion on this page. When you are on a specific project's detail page, the AI knows exactly which project you are looking at. You can ask for help refining the **Elevator Pitch** or identifying potential **Success Metrics** based on your strategic theme.
