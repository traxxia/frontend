# Audit Trail

The **Audit Trail** provides a detailed log of important system events across the platform.

It helps administrators monitor security and user activity.

---

## Event Records

Each audit record includes:

- Event type
- Timestamp
- User
- Description
- Event Details

![Audit Trail Dashboard](/academy-screenshots/admin-panel/audit-trail.png)

Examples of recorded events include:

- Successful login
- Failed login attempts
- User logout
- Administrative actions

---

## Viewing Event Details

Each event contains a **View** button that opens a detailed event log.

This modal shows:

- User information
- Email
- Role
- Company
- Event result
- System details and more

![Audit Event Details](/academy-screenshots/admin-panel/audit-details.png)

The Audit Trail helps ensure transparency and security across the platform.
