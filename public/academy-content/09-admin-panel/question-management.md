# Question Management

The **Question Management** section allows Super Admins to manage the AI questionnaire used in the Traxxia strategy analysis process.

These questions are used to generate strategic insights and analysis.
 
## Question List

The question table displays:

- Question number
- Question text
- Strategic framework usage
- Severity level
- Available actions

![Question Management](/academy-screenshots/admin-panel/question-management.png)
 

## Question Actions

Administrators can perform several actions on each question:

- **Edit** – Modify the question text or settings.
- **Delete** – Remove outdated questions.
- **Details** – View additional question information.
 
## Adding a New Question

Super Admins can create new questions by clicking the **Add Question** button.
