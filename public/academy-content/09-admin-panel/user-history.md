# User History

The **User History** section allows Super Admins to review activity records for individual users.

This feature helps administrators understand how users interact with the Traxxia platform.
 
## User History Table

The User History page lists:

- User Name
- Role
- Company
- Joined Date

Each row includes a **View History** button.

![User History Table](/academy-screenshots/admin-panel/user-history-list-admin.png)
 

## Viewing Detailed User History

When clicking **View History**, a detailed modal appears displaying the user's activity.

The modal contains multiple sections such as:

- Businesses
- Conversations
- Insights (6 Cs)
- Strategic Framework data
