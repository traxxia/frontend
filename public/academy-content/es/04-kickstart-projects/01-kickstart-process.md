# El Proceso Kickstart (Resumen)

La transición de la planificación estratégica a la ejecución es fluida con el motor Kickstart de Traxxia. Esta página ofrece una visión general del proceso.

## 1. El Punto de Entrada: Resumen Ejecutivo
Después de completar el **Onboarding PMF**, verás tu **Resumen Ejecutivo**. Este es el primer lugar donde se identifican tus **3–5 Principales Prioridades**. Cada una de estas prioridades representa una línea de trabajo estratégica lista para su ejecución.

## 2. La Pestaña Prioridades
Para actuar sobre estas prioridades, navega a la pestaña **Prioridades**. Aquí puedes:
- **Explorar** todas las prioridades estratégicas identificadas.
- **Seleccionar** cuáles ejecutar usando las casillas de verificación.
- **Iniciar** la creación de proyectos reales.

## 3. Restricciones de Plan y Rol
- **Usuarios Advanced**: Tienen acceso completo al botón cohete **"Iniciar Proyectos"**.
- **Usuarios Essential**: Verán un aviso **"Actualizar para Iniciar"**.
- **Administradores**: Son los únicos que pueden activar el flujo de creación de proyectos.
