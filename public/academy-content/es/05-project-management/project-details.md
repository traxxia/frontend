# Detalles del Proyecto y Visualización

Al hacer clic en cualquier tarjeta de proyecto o usar el menú de edición se abre la página de **Detalles del Proyecto**. Aquí es donde documentas la "Apuesta Estratégica" específica que estás realizando.

## 1. Núcleo Estratégico
Esta sección define el corazón del proyecto:
- **Elevator Pitch**: Un resumen del proyecto en una sola frase.
- **Por qué es importante**: El valor empresarial y la justificación.
- **Decisión Estratégica**: La apuesta principal que estás realizando.
- **Suposiciones Clave Evaluadas**: Una lista de cosas que deben ser ciertas para que este proyecto tenga éxito.
- **Criterios de Éxito (Continuar Si)**: Métricas que indican que el proyecto está funcionando.
- **Criterios de Cancelación (Detener Si)**: Señales de advertencia que indican que el proyecto debe abandonarse.

## 2. Contexto Estratégico
Estos campos son utilizados por la IA para ayudar a construir tu matriz de priorización:
- **Impacto**: El beneficio potencial (Alto, Medio, Bajo).
- **Esfuerzo**: Los recursos estimados necesarios (Pequeño, Medio, Grande).
- **Riesgo**: El nivel de incertidumbre.
- **Tema Estratégico**: Alineación por categoría (Crecimiento, Eficiencia, Innovación, etc.).

## 3. Metadatos Adicionales
Para una planificación más avanzada, puedes documentar:
- **Restricciones**: Elementos "no negociables" del proyecto.
- **Dependencias**: Otros proyectos o recursos necesarios.
- **Fuera de Alcance**: Límites claros para evitar la expansión del alcance.
- **Cronograma y Presupuesto Estimado**: Restricciones lógicas para la iniciativa.

## 4. Contexto del Asistente de IA
La barra lateral de IA es tu acompañante constante en esta página. Cuando estás en la página de detalles de un proyecto específico, la IA sabe exactamente qué proyecto estás viendo. Puedes pedir ayuda para mejorar el **Elevator Pitch** o identificar posibles **Métricas de Éxito** basadas en tu tema estratégico.
