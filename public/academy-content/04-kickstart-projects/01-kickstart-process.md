# The Kickstart Process (Overview)

Transitioning from strategic planning to execution is seamless with Traxxia's Kickstart engine. This page provides a high-level overview of the process.

## 1. The Entry Point: Executive Summary
After you finish the **PMF Onboarding**, you'll see your **Executive Summary**. This is the first place where your **Top 3-5 Priorities** are identified. Each of these priorities represents a strategic workstream ready for execution.

## 2. The Priorities Tab
To act on these priorities, navigate to the **Priorities** tab. Here, you can:
- **Browse** all identified strategic priorities.
- **Select** which ones to execute using the checkboxes.
- **Kickstart** the creation of actual projects.

## 3. Plan & Role Constraints
- **Advanced Users**: Have full access to the "Kickstart Projects" rocket button.
- **Essential Users**: Will see an "Upgrade to Kickstart" prompt.
- **Admins**: Are the only ones who can trigger the project creation flow.
